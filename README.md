# BrokenCray0n Idevice

## How do I install these formulae?
`brew install brokencray0n/idevice/<formula>`

Or `brew tap brokencray0n/idevice` and then `brew install <formula>`.

Or install via URL (which will not receive updates):

```
brew install https://raw.githubusercontent.com/BrokenCray0n/homebrew-idevice/master/Formula/<formula>.rb
```

## Documentation
`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
